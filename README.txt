Dojo de Mensageria

Esse dojo tem o objetivo de demonstrar o uso e características da mensageria.
Iremos incrementar pouco a pouco a aplicação, para demonstrar cenários mais simples para mais complexos.

Abaixo os passos a serem desenvolvidos. Ao finalizar cada item, favor commitar as mudanças. :)

1 - Criar um Produtor/Consumidor que envie um número e verifique se ele é primo ou não. 
	Exemplo: Produtor->10;Consumidor->N, Produtor->11;Consumidor->S

2 - Altere o programa para ele enviar um intervalo de números, e o consumidor mostrar quais números primos existem no intervalo.
	Exemplo: Produtor->10,20;Consumidor->11,13,17,19

3 - Altere o programa para enviar o intervalo de números de forma segmentada em 3 partes. 
	Exemplo: input:10,20->(Produtor->10,13;14-17,18-20);Consumidor->(11,13;17;19)

4 - Altere o programa para que o Cliente receba o resultado e escreva na saida do seu console. Utilize a arquitetura RPC para alcançar esse objetivo.
